## What Is *Out of Scope*

The following are intentionally **not accepted** in this repository:

- Requests to open-source the EVE core
- Reverse-engineering attempts
- PRs that attempt to replicate internal core logic
- Contributions that bypass safety or control layers
- Feature demands that conflict with EVE‚Äôs design principles

EVE prioritizes **correctness, traceability and human accountability** over speed or convenience.

---

## How to Contribute

1. **Open an Issue first**
   - Describe your idea, suggestion or question
   - Explain *why* it matters

2. **Discuss before implementing**
   - Most contributions are design- or documentation-driven
   - This avoids wasted work

3. **Submit Pull Requests**
   - Keep changes focused and minimal
   - Reference the related issue
   - Follow existing style and structure

---

## Guiding Principles

All contributions should align with EVE‚Äôs core philosophy:

- Deterministic behavior
- Explicit execution boundaries
- Observable and debuggable systems
- Separation of analysis and action
- Human-in-the-loop control

If a proposal violates these principles, it will not be merged.
